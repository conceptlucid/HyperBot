# HyperBot

Your personal AI that controls any computer. Any OS. Any Platform.

## Quick Install

```bash
curl -sL hyperbot.sh | bash
```

## Development

```bash
# Cloud app
cd hyperbot-cloud
npm install
npm run dev
```

## Architecture

- **hyperbot.sh** - One-line installer
- **hyperbot-cloud/** - Next.js cloud dashboard (myhyperbot.com)
- **hyperbot-agent/** - The agent that runs on machines (coming soon)
